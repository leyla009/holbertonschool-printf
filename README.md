# holbertonschool-printf

This repository contains the Holberton School `_printf` project.

## Task 0: _printf

- Prototype: `int _printf(const char *format, ...);`
- Handles conversion specifiers: `%c`, `%s`, `%%`
- Uses `write` to output characters to stdout
- Returns the number of characters printed
- Uses variadic functions: `va_start`, `va_arg`, `va_end`

## Files

- `main.h` : header file
- `0-printf.c` : implementation of `_printf`
